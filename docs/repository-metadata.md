# Repository Metadata Checklist

Use this checklist after creating the remote repository.

## Suggested GitHub Description

`Local AI agent desktop app powered by OpenClaw, Electron, and provider-ready runtime packaging.`

## Suggested Topics

- `ai-agent`
- `desktop-app`
- `electron`
- `local-ai`
- `llm`
- `ollama`
- `openclaw`
- `react`
- `typescript`
- `productivity`

## Suggested Social Preview

- Size: `1280x640`
- Visual: app screenshot or product hero image
- Include: `CatClaw`, `Local AI Agent Desktop`, `Powered by OpenClaw`

## Releases

- Publish desktop installers through GitHub Releases.
- Do not commit `.dmg`, `.zip`, `.exe`, or unpacked app bundles into the main branch.
