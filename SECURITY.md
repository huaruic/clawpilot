# Security Policy

## Reporting A Vulnerability

Please do not open a public issue for security-sensitive bugs.

Send a private report with:

- affected version
- impact summary
- reproduction steps
- logs or screenshots if available

Until a dedicated security mailbox is published, contact the maintainers through a private GitHub security advisory or the repository owner contact listed in GitHub.

## Runtime Notes

CatClaw packages a local OpenClaw runtime for desktop distribution. Security fixes in:

- OpenClaw
- Electron
- Node.js

should be treated as update priority items.
